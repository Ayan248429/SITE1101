Personal Website — Starter



Customize
1. Replace `USERNAME`, `REPO`, and `Your Name` with your details in the HTML files.
2. Add `images/profile.jpg` and project screenshots to the `images/` folder.
3. Update project descriptions and repository links in `projects.html`.

Make repository public and deploy (simple):
- Create a GitHub repository, push this folder, then enable GitHub Pages from the repo settings (use `main` branch or `gh-pages`).

Quick local preview (any OS):
- Open `index.html` in a browser, or run a local server, for example with Python:

```bash
python -m http.server 8000
```
